## 项目高级概览

**内容播放列表播放器** — 一个基于 **Electron + Vite** 的桌面应用，用于打开本地文件夹、扫描媒体文件并进行播放。

### 技术栈

| 层级 | 技术 |
|------|------|
| 桌面框架 | Electron 41 |
| 前端构建 | Vite 8 |
| 打包发布 | electron-builder (NSIS 安装包) |
| 语言 | 原生 JavaScript（无框架） |

### 进程架构

```
┌─ 主进程 (main.js) ─────────────────────┐
│  - 创建 BrowserWindow                   │
│  - IPC: select-directory / scan-directory│
│  - 加载 dist/index.html                 │
└──────────┬──────────────────────────────┘
           │ contextBridge (preload.js)
           │ 暴露 window.api
┌──────────▼──────────────────────────────┐
│ 渲染进程 (src/)                          │
│  - index.html  → 单页界面（侧边栏+播放区）│
│  - main.js     → 播放列表、媒体切换、导航  │
│  - style.css   → 暗色主题               │
└─────────────────────────────────────────┘
```

### 核心功能

- **打开文件夹** → 扫描顶层媒体文件（视频/音频/图片）
- **播放列表侧边栏** → 展示扫描到的文件列表
- **主区域播放** → 根据文件类型自动切换 video/audio/img
- **导航** → 上一个/下一个、自动播放、键盘快捷键
- 媒体通过 `file:///` 绝对路径加载，安全隔离（contextIsolation 开启，nodeIntegration 关闭）

### 项目结构

```
content-player/
├── main.js          # Electron 主进程
├── preload.js       # contextBridge 桥接
├── vite.config.js   # Vite 配置 (root=src, out=../dist)
├── src/
│   ├── index.html   # 单页界面
│   ├── main.js      # 渲染逻辑
│   └── style.css    # 样式
├── dist/            # Vite 构建输出
└── package.json
```

零外部运行时依赖，仅有 `electron`、`electron-builder`、`vite` 三个开发依赖。项目结构非常精简。
