# 内容播放器 — 产品需求文档（PRD）

## 项目概述

基于 Electron + Vite 的桌面媒体播放列表播放器，打开本地文件夹后自动扫描媒体文件，支持视频、音频、图片的顺序播放。

## 核心功能

### 文件夹选择与扫描

- 点击「选择文件夹」按钮选择本地目录
- 自动扫描目录中的媒体文件（视频/音频/图片）
- 支持的格式：
  - 视频：mp4, webm, ogg, ogv, avi, mkv, mov
  - 音频：mp3, wav, flac, aac, m4a, wma
  - 图片：jpg, jpeg, png, gif, bmp, webp, svg

### 播放列表

- 左侧显示播放列表，点击即播放对应文件
- 当前播放项高亮显示

### 播放控制

- 视频/音频播放结束后自动播放下一个
- 图片展示指定时间后自动切换（默认 5 秒）
- 上一个/下一个按钮手动切换
- 可关闭自动播放
- 键盘快捷键：
  - 方向键（左/上）：上一个
  - 方向键（右/下）：下一个
  - 空格键：暂停/播放

## 界面要求

- 除去标准菜单栏，保持界面简洁
- 暗色主题
- UI 语言为中文

## 选项菜单

- 顶部栏右侧「选项」按钮，点击弹出下拉菜单
- 选项内容：
  - 视频/音频重复播放次数（默认 1 次）
  - 图片展示时间（默认 5 秒）

## 数据持久化

- 自动保存当前选择的文件夹路径，下次打开应用后自动加载
- 自动保存所有选项参数（重复播放次数、图片展示时间、自动播放开关），下次打开应用后自动恢复

## 打包发布

- 使用 electron-builder 打包为 Windows 安装包（NSIS）
- `npm run pack` — 生成免安装目录
- `npm run dist` — 生成安装包
